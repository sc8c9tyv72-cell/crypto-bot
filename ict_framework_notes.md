## 第一部分：加密貨幣日內交易（狙擊手模式）

### 身份設定
- 目標：賺取日內波幅現金流
- 本金：HK$100,000（約 12,800 USDT）
- 核心風控：單筆固定風險 50 USDT
- 核心時區：1H（定方向）→ 15M（找戰場/確認）→ 3M（執行/優化）

### 開市三問（Routine Check）
1. **方向確認 (Bias)**：1H 圖表上，價格想去掃上方 BSL 還是下方 SSL 的流動性？
2. **層級陷阱 (Hierarchy Trap)**：當前 15M 阻力位上方，是否還有未測試的 1H 關鍵位？（若有，當前 15M 極大機率是誘餌，不可做空）
3. **風險事件 (News)**：接下來一小時有無重大數據公佈？

## 第二章：狙擊手模式（頂底反轉）— 雙版本

### 適用場景
價格觸碰 **1H/15M 大級別關鍵位 (POI)**

### 版本 A：激進預判版（Aggressive 3M Entry）
- 適用：極強的關鍵位（如日線/週線級別），或出現巨量獵殺針（SFP）
- 觸發：價格進入 15M 關鍵區 → 立刻切換至 3M
- 確認（3M）：等待 3M K線實體收盤 跌破/突破 3M 前一個結構低點/高點 = **3M MSS**
- 部署：針對 3M 的跌破波段拉斐波那契
- 入場：在 3M 溢價區內尋找 **3M FVG** 掛單
- **止損（SL）**：雖然是用 3M 入場，但強烈建議 SL 依然放在 **15M 的結構高點之外**
  - 原因：3M 的高點很容易被「二次測試」掃掉。用 15M 防守，用 3M 入場，勝率與防守兼備

### 版本 B：穩健確認版（Robust 15M Confirmation）
- 適用：普通的 15M 阻力位，或震盪盤整行情。假突破風險高時使用
- 觸發：價格進入 15M 關鍵區 → 忍手，保持在 15M 圖
- 確認（15M）：必須等待 15M K線實體收盤 跌破/突破 15M 前一個關鍵低點/高點（Swing Low/High）= **ChoCH**
  - 注意：單純插針跌破不算，必須實體收低
- 部署：針對「導致結構破壞」的 15M 波段拉斐波那契
- 入場：在 15M 溢價區（0.618-0.79）內尋找 **OB 或 FVG**
- **止損（SL）**：放在該波段的最高點（獵殺針尖）之外

### 決策心法：A 還是 B？
1. **看動能（Momentum）**：
   - 噴射式衝入關鍵位，瞬間留下超級長避雷針（Long Wick）→ 用版本 A（激進）
   - 緩慢爬升進入關鍵位，磨磨蹭蹭 → 用版本 B（穩健）
2. **看層級（Hierarchy）**：
   - 日線（Daily）級別阻力位 → 用版本 A（大級別反應通常劇烈）
   - 只是 15M 級別 → 用版本 B（小級別容易失效）

## 3. 進階：滾雪球入場法（Zone Scaling）— 30/70 法則
- 核心邏輯：分兩注入場，但共用同一個結構性止損（15M Swing High）
- **第一注（偵察兵）**：3M 出現 MSS 時，市價或淺回調入場。注碼：總風險的 30%（15 USDT）
- **第二注（狙擊手）**：掛單（Limit）在 15M 極值 OB 或 斐波那契 0.79 深回調位。注碼：總風險的 70%（35 USDT）

## 第三章：衝浪者模式（順勢中繼）

### 路徑 A：淺回調（強勢趨勢）— 3M 直擊
- 場景：消息面驅動暴跌，或 K線沿著 EMA 線狂衝。回調好弱，根本據唔到 15M 嘅關鍵位
- 操作：
  1. 識別強勢（15M 圖全是紅燭，中間只有極短綠燭）→ 切去 3M
  2. 在 3M，針對最近一小波下跌拉斐波那契
  3. 只要價格回彈進入 3M 溢價區（0.5 以上）→ 掛單在 3M 的 FVG 或 Breaker 位置
  4. 止損：放在 3M 回調高點上方

### 路徑 B：深回調（標準結構）— 3M 結構確認
- 場景：正常趨勢運行。價格跌深了反彈，或撞到了下方的 15M 支撐位
- 操作：
  1. 等待到位（價格反彈，觸碰了上方的 15M OB 或 S/R Flip）
  2. 觀察微觀結構：3M 的走勢應該是「向上」的（因為正在回調）→ 盯著 3M 的 Higher Low（小支撐）
  3. 確認轉勢（MSS）：等待一根 3M K線 實體跌破 剛才那個 3M 小支撐（MSS）
  4. 入場：在這個新形成的 3M 下跌波段中，找 FVG 入場
  5. 止損：放在 15M 關鍵位上方（防守大級別）

## SL 邏輯總結（加密貨幣日內）
- **版本 A（3M 入場）**：SL 放在 **15M 結構高點（獵殺針尖）之外**
- **版本 B（15M 入場）**：SL 放在 **該波段最高點（獵殺針尖）之外**
- **衝浪者路徑 A（3M 直擊）**：SL 放在 **3M 回調高點上方**
- **衝浪者路徑 B（深回調）**：SL 放在 **15M 關鍵位上方**

**共同原則：SL = 關鍵位置的 Low/High 外側（不是固定 ATR，是結構性止損）**

## 關鍵位識別優先級（加密貨幣日內）
1. 1H 結構高/低點（Swing High/Low）= 方向轉折點
2. 1H OB/FVG（訂單塊/公平價值缺口）
3. 15M OB/FVG/IFVG（入場精確位）
4. BSL/SSL（上方/下方流動性）
5. PWH/PWL/PDH/PDL/WO/DO（週日關鍵位）
6. FIB（輔助，0.5/0.618/0.705/0.786）

## 入場確認（3M 低時區）
- 等待價格進入 15M 關鍵區
- 切換至 3M，等待 3M MSS（Market Structure Shift）
- 3M MSS = 3M K線實體收盤 跌破/突破 3M 前一個結構低/高點
- 在 3M FVG 掛單入場
